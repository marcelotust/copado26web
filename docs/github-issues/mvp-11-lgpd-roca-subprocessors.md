## Contexto

LGPD exige transparência sobre subprocessadores, retenção e bases legais além das páginas públicas.

## Escopo

- Documento em `docs/` (registro de tratamento / ROPA simplificado): categorias de dados, finalidades, prazos de retenção, subprocessadores (Supabase, Vercel, Google, Sentry), transferência internacional se aplicável.
- Procedimento interno resumido para solicitações de titulares e incidentes.
- Alinhar com política de privacidade publicada na issue de termos.

## Fora de escopo

- Dados de câmera/OCR.

## Critérios de aceite

- [ ] Doc versionado e referenciado na política de privacidade.
- [ ] Retenção de logs Sentry/analytics documentada.
- [ ] Responsável/contato para privacidade indicado.
